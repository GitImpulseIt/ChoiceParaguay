# Instructions Claude - Projet ChoiceParaguay

> **Ce fichier contient uniquement des instructions pour Claude.** Il ne doit pas contenir de documentation projet. Il ne peut être modifié que sur demande explicite de l'utilisateur.

## Documentation

- Toute la documentation du projet se trouve dans `docs/` et dans aucun autre répertoire.
- Ne jamais créer de fichiers de documentation en dehors de `docs/`.

## Spécifications

- Le projet ChoiceParaguay doit respecter les spécifications contenues dans `docs/CHOICE.pptx`.
- Les choix artistiques (couleurs, typographies, visuels, ambiance) doivent respecter le thème du compte Instagram présent dans `docs/insta/`.

## Stack technique

- **PHP / Twig / Tailwind CSS / jQuery**
- Toujours utiliser les toutes dernières versions disponibles de chaque technologie.

## Règles de code

- **Ne jamais mélanger les langages dans un même fichier.** Chaque type de contenu a son propre fichier :
  - `.php` pour le PHP
  - `.twig` pour les templates Twig
  - `.css` pour les styles
  - `.js` pour le JavaScript
  - Les assets (images, polices, etc.) dans leurs répertoires dédiés

## Git

- Toujours commit et push après chaque changement.
